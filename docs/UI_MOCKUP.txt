- Real-time toggle between manual and HA coordinates
- Input validation with proper ranges
- Display of current HA coordinates when manual mode disabled
- Responsive layout that works on mobile and desktop
- Consistent styling with other Smart Irrigation cards
- Localized text in 8 languages
